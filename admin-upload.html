Accepted File Formats

You can upload files in the following formats:

  • JSON (Note: any image type should work): 
    [ 
    {
        "id": 1,
        "name": "Product1",
        "description": "Description1",
        "category": "Category1",
        "image": "path/image.png",
        "price": 10.99
    }
]
  • CSV and Text: 
    "id,name,description,category,image,price"
